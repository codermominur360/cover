<strong>Contact Form 7 Store To DB Lite</strong> - is the extension/addon to the contact form 7 plugin used to store all the submission entries submitted via the contact form 7
without losing all the data including the attachments. Entries from the contact form 7 plugin will be stored safely even if the form failed to get submitted or any of your email get lost, deleted or removed by mistake.

<strong>Protect, Manage and store the Contact Form 7 entries of your business which are important to you using the plugin.</strong>

= Features: =

* <strong>Custom Post Type Entries</strong>
 	- Custom post type for easy handling of the contact form 7 entries.
* <strong>Attachment Storage in Default WordPress Gallery</strong>
 	- When the attachment is submitted through contact form 7 plugin, plugin store the attachment in the default WordPress Gallery.
* <strong>Delete Attachment While Deleting Entries</strong>
	- Ability to set if attachment related with the specific post will also be deleted.
* <strong>Enable Direct Delete of Entries Post</strong>
	- Ability to set if trashing the entries will skip trash and permanently delete the entries.

== Installation ==
1. Unzip contact-form-7-store-to-db-lite.zip
2. Upload all the files to the /wp-content/plugins/contact-form-7-store-to-db-lite/
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on CF7 Entries in WordPress left admin menu.
